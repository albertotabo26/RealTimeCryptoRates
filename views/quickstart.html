RealTimeCryptoRates Quick Start Guide

1. Obtaining an API Key

Visit CoinAPI and sign up to obtain an API key.

2. Configuring the .env File

Create a .env file in the root directory of the project and add your API key as shown below:
API_KEY=your_coinapi_key_here

3. Running the Application

Run npm run build to build the project, followed by npm start to start the server.

4. Accessing the App

Open your browser and visit localhost:3000 to view the app.

5. Using the App

Select a cryptocurrency and a traditional currency, then click the 'Get Exchange Rate' button to view the exchange rate.
